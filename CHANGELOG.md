# Change Log

All notable changes to the "youtube-mcp-tools" extension will be documented in this file.

Check [Keep a Changelog](http://keepachangelog.com/) for recommendations on how to structure this file.

## [0.1.0] - 2026-01-27

### Added

- YouTube video search with tree view display
- Video analysis with summaries, concepts, and quality assessment
- Transcript extraction with timestamps
- Flashcard generation from video content
- Quota monitoring for YouTube API usage
- Support for API key via VS Code settings or .env file
- Activity bar with Search Results, Recent Videos, and Flashcards views
- Status bar indicator for extension state
